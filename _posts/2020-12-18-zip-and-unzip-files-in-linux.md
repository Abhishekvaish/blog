zip, unzip file using gzip and compress and decompress folder/directory using tar 

#### Zip a file
```sh
gzip filename
```
#### Unzip a file
```sh
gunzip filename.gz
```
#### Compress a folder
```sh
tar cvf name_of_compressed_file.tar  folder_name_or_list_of_files
```
#### Extract a file
```sh
tar xvf filename.tar
```
